##  Typescript language

Typescript는 해당 변수나 함수에 대한 변수 형태를 지정해줄 수 있습니다.

```Typescript
let hello:string = "hello";
// 위와 같이 작성하는것을 typing이라고 부릅니다.
```

아래와 같이 인터페이스를 설정할 수 있습니다.

```Typescript
interface Ihuman{
  name: string;
  age?:number;
  hungry:boolean;
}
```
